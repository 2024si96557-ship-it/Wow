# Women @ Work — Give to Gain

A responsive, accessible website celebrating women's journeys and enabling mentorship connections for International Women's Day 2026.

## 🌟 Features

- **Hero Section** with animated elements and International Women's Day theme
- **5 Category Tiles** for different professional domains
- **9 Mentor Profiles** with contact options
- **Story Submission Form** with validation
- **Monthly Spotlight** featuring inspiring women leaders
- **Resource Library** with professional development tools
- **Community Section** for ongoing engagement

## 🎨 Design

- **Colors**: Purple & Green (International Women's Day colors)
- **Responsive**: Mobile-first design with smooth animations
- **Accessible**: WCAG AA compliant with proper ARIA labels

## 🚀 Quick Start

Simply open `index.html` in any modern web browser.

## 📁 Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Dynamic content and interactions

## 🌐 Deploy to Share

### Option 1: GitHub Pages (Recommended - Free)
1. Create a new repository on GitHub
2. Push these files
3. Go to Settings → Pages → Enable GitHub Pages
4. Share the URL with your team!

### Option 2: Netlify (Easiest - Drag & Drop)
1. Go to [netlify.com](https://netlify.com)
2. Drag the entire WOW folder
3. Get instant public URL

### Option 3: Azure Static Web Apps
1. Use Azure portal or CLI
2. Deploy this folder as a static web app
3. Perfect for enterprise sharing

## 📝 Customization

Edit the data in `script.js`:
- `categories` array - Category information
- `mentors` array - Mentor profiles
- `resources` array - Resource links
- `spotlight` object - Monthly featured profile
